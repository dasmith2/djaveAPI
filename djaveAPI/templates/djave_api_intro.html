Welcome to the API

It speaks JSON and uses basic authentication.

You can read and write any important object on demand using good old fashioned GET and POST requests.

You can subscribe to webhooks and we'll let you know when any important object changes in any way.

You can get your very own "API Key username" and "API Key password" and tell us where to send webhooks under the "Settings" link in the upper right hand corner when you're logged in.

The API is rate limited to one request a second. If you exceed the rate limit, you'll get a response with status code 429.
